Cinema Express App

Description

This web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up or deregister, update their personal information, and create a list of their favorite movies.


Supported Requests

Request URL HTTP Method Request Data Format Response Data Format
Allows users to register /users POST None JSON object holding data about the user to add.
Allows users to update their information /users/:id PUT JSON object holding data to update. JSON oject holding data about the user that was updated.
Allows users to ADD movies to favorite movies /users/:id/:movieTitle POST None Text message indicating the selected movie was successfully added to the user's favorites list.
Allows users to DELETE movies from favorite movies /users/:id/movieTitle DELETE None Text indicating the selected movie was successfully removed from the user's favorites list.
Allows users to deregister /users/:id DELETE None Text indicating the user's profile has been deleted.
Get a list of ALL movies /movies GET None An array of JSON objects holding data on ALL movies in the database.
Get data on a single movie by title /movies/:title GET None JSON object holding data about the requested movie.
Get data about a genre by name or title /movies/genre/:genreName GET None JSON object holding data about the requested genre.
Get data about a director by name /movies/directors/:directorName GET None JSON object holding data about the requested director.
